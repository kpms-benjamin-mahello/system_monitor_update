Docker

The Docker screen lists each Docker container that is on the DataCollector image. Each container contains a service that is executed on the DataCollector.

The table shows the following details for each container:

Name

 Name of the container

ID

 ID of the container
Status Status of the container and duration how long it is in this status

CPU load

 Current processor utilization

RAM usage

 Currently used RAM

Version

 Version of the container

Log

 Click on this button to get a file with a current log

In the top left corner, you can select how many containers you want to display on one page.

A search is available in the top right corner.

If there are more containers than columns on one page, you can click through the pages of the column by selecting "Next" or "Previous".
