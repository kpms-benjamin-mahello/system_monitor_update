System

The System screen provides details about the operating system and general parameters.

The table shows the following details:

Computer name This indicates the name of the computer
Local time Currently set time on the DataCollector
Uptime Duration in seconds since starting the DataCollector
IP IP address of the DataCollector
CPU load Total processor utilization
RAM usage RAM currently by all processes
Disk space available Disk space that is free for usage
OS version Version of the operating system

 

In the top left corner, you can select how many containers you want to display on one page.

A search is available in the top right corner.

If there are more containers than columns on one page, you can click through the pages of the column by selecting "Next" or "Previous".
